Prepositional genitive

Advanced input() handling is used to read input directly into several variables, for example:

x, y = input().split()

Use it to print the next message with the two input values: "x of y"


Sample Input:
state Massachusetts

Sample Output:
state of Massachusetts


Sample Input:
Queen Scots

Sample Output:
Queen of Scots
